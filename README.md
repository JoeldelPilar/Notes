# About NOTES

This project is a part of the course API development at Medieinstitutet, Stockholm, Sweden <br>

## Get up and running

### backend:

cd into the map "backend"

> `npm install`

to connect to the database

> `nodemon start`

## TECH STACK

### Frontend:

- Vue.js 3
- Pinia & vue router
- TinyMCE for vue 3
- CSS

### Backend

- Express
- Cors
- ENV
- SQL querys

### DB

- MySQL
- MAMP
- PHPmyAdmin GUI
